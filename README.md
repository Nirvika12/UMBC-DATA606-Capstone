# UMBC DATA606 Capstone – Predicting Student Startup Success

**Author:** Nirvika Rajendra  
---

## Project Overview
Predicting the success of student-led startups using data from 40 academic institutions (2019–2023).  
Key drivers include **funding, innovation, mentorship, incubation, and business model clarity**.

---

## Dataset
- Source: [Kaggle – Student Startup Success](https://www.kaggle.com/datasets/ziya07/student-startup-success-dataset/data)  
- Target: `success_label` (1 = Success, 0 = Failure)  

---
